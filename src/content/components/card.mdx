# Card

Flexible container components for organizing and displaying related content.

<ComponentStatus status="beta" />

## Overview

Cards group related information and actions together in a contained area. They're useful for organizing content in a scannable format.

<ComponentPreview
  code={`<elvt-card>
  <h3 slot="header">Card Title</h3>
  <p>This is the card content area where you can place any content.</p>
  <elvt-button slot="actions" tone="primary">Action</elvt-button>
</elvt-card>`}
/>
