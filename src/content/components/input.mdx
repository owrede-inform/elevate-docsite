# Input

Text input fields for user data entry with built-in validation support.

<ComponentStatus status="stable" />

## Overview

Input components allow users to enter text, numbers, and other data. They provide consistent styling and validation across your application.

<ComponentPreview
  code={`<elvt-input placeholder="Enter your name" label="Name"></elvt-input>
<elvt-input type="email" placeholder="Enter your email" label="Email"></elvt-input>
<elvt-input type="password" placeholder="Enter password" label="Password"></elvt-input>`}
/>
