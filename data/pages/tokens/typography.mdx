## Typography Architecture

### Primitive Typography Tokens
Raw typography values that form the foundation of the ELEVATE type system.

#### Font Families
```css
--elvt-primitives-font-family-default: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif
--elvt-primitives-font-family-mono: 'Roboto Mono', 'Fira Code', Monaco, 'Cascadia Code', monospace
```

#### Font Weights
```css
--elvt-primitives-font-weight-100: 100  /* Thin */
--elvt-primitives-font-weight-200: 200  /* Extra Light */
--elvt-primitives-font-weight-300: 300  /* Light */
--elvt-primitives-font-weight-400: 400  /* Regular */
--elvt-primitives-font-weight-500: 500  /* Medium */
--elvt-primitives-font-weight-600: 600  /* Semi Bold */
--elvt-primitives-font-weight-700: 700  /* Bold */
--elvt-primitives-font-weight-800: 800  /* Extra Bold */
--elvt-primitives-font-weight-900: 900  /* Black */
```

## Usage Examples

### Using Typography Tokens in CSS

```css
/* Apply headline styles */
h1 {
  font-family: var(--elvt-primitives-font-family-default);
  font-size: var(--elvt-type-measures-headline-size-l);
  font-weight: var(--elvt-primitives-font-weight-600);
  line-height: var(--elvt-type-measures-headline-lineHeight-l);
}

/* Apply content styles */
p {
  font-family: var(--elvt-primitives-font-family-default);
  font-size: var(--elvt-type-measures-content-size-l);
  font-weight: var(--elvt-primitives-font-weight-400);
  line-height: var(--elvt-type-measures-content-lineHeight-l);
}

/* Apply label styles */

  font-weight: var(--elvt-primitives-font-weight-500);

}
```

### Inter Variable Font Features

ELEVATE typography leverages Inter's variable font capabilities:

```css
/* Optimize for different text sizes using optical sizing */
.large-text {
  font-variation-settings: 'opsz' 32;
  font-size: var(--elvt-type-measures-display-size-m);
}

.body-text {
  font-variation-settings: 'opsz' 16;
  font-size: var(--elvt-type-measures-content-size-l);
}

.small-ui-text {
  font-variation-settings: 'opsz' 14;
  font-size: var(--elvt-type-measures-label-size-m);
}

/* Use variable weight axis for smoother weight transitions */
.interactive-text {
  font-variation-settings: 'wght' 450;
  transition: font-variation-settings 0.2s ease;
}

.interactive-text:hover {
  font-variation-settings: 'wght' 500;
}
```

## Accessibility

ELEVATE typography tokens ensure accessibility through:

- **Sufficient contrast**: All text meets WCAG 2.1 AA contrast requirements
- **Readable line heights**: 1.4+ line height for body text, appropriate spacing for headings
- **Scalable units**: Uses rem units for better user zoom support
- **Clear hierarchy**: Distinct size and weight differences between heading levels
- **Optimal reading**: Inter font optimized for screen reading at all sizes

## Best Practices

1. **Use semantic scales**: Choose sizes appropriate for content hierarchy
2. **Maintain consistency**: Use the same token for similar content types
3. **Consider reading distance**: Larger sizes for hero content, readable sizes for body text  
4. **Test responsively**: Verify typography works across different screen sizes
5. **Leverage variable fonts**: Use Inter's optical sizing for better rendering
6. **Follow accessibility**: Ensure sufficient contrast and spacing
